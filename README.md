# Facebook-Cleaner
Facebook Cleaner Chrome Extension


# Functions:
<code>
Clean all Reactions,<br>
Comments, Search History, 
Watched Video History,<br>
Unfollowed History,<br>
Stories Reactions,<br>
Stories Archives,<br>
Sent Friend Request,<br>
Voted Polls,<br>
Hidden Posts,<br>
Photos,<br>
Videos,<br>
Saved Posts.
</code>
